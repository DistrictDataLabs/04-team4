Goal

To provide a visualization of a graph that supports representations of large scale interactions and chages in the graph over time. We plan to use interactive graphs that allow the user to adjust a start and end time for the representation and see the chages in the graph.

"Goal" Data

Baleen is a tool for ingesting formal natural language data from the discourse of professional and amateur writers: e.g. bloggers and news outlets. Rather than performing web scraping, Baleen focuses on data ingestion through the use of RSS feeds. It performs as much raw data collection as it can, saving data into a Mongo document store.

As of April 19, 2016, the corpus ingested 373 feeds containing 180,342 posts in 13 categories.

"Toy" Data

Les Miserables character interactions, 80 characters in 356 chapters:
  • Data source 1: Les Miserables character by chapter
  • Data source 2: Les Miserables character groups
Chapter designations are similar to dates and can be used to define a range. Graphs will show interactions within defined ranges of chapters.
